Chroma For Cart

Chroma For Cart or CFC is a server made out of JS that contacts the Razer Chroma REST API while getting IN-GAME info from supported mods to be able to display game information in RGB.

Though the infrastructure allows for multiple games, for v1 and v2 we will focuse on the core engine of the project (the CFC server).

Has error detection and a log system so you are able to know both response time and error cause for solo bug fixing.

-

Only supports Razer products that are compatible with synapse 4 and the Chroma SDK.
